#  Employee Salary Prediction using Streamlit

##  Overview
A machine learning web app that predicts **employee salaries** based on experience, education, and role using a **Random Forest Regressor**.  
Built with **Python, Scikit-learn, and Streamlit**.

---

##  Features
- Data cleaning & preprocessing with Pandas  
- Salary prediction using RandomForestRegressor  
- Model saved with Pickle  
- Interactive Streamlit interface  

---

##  Tech Stack
- Python, Pandas, NumPy  
- Scikit-learn, Matplotlib  
- Streamlit
